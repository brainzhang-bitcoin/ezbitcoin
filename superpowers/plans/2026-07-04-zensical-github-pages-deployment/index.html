Zensical Static Site and GitHub Pages Deployment Implementation Plan¶

For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.

Goal: Configure Zensical static site generator for EzBitcoin docs and setup automated GitHub Pages deployment via GitHub Actions with custom domain ezbitcoin.brainz.fun.

Architecture: Install zensical, configure site metadata/theme/nav in zensical.toml, and write a deployment YAML workflow.

Tech Stack: Python 3.12, Zensical, GitHub Actions.

Global Constraints¶

  • Target URL: https://ezbitcoin.brainz.fun/
  • Color scheme: Classic Indigo (primary & accent)
  • Flattened navigation structure (Home, Beginners Guide, Technical Core, Networking & Mining, Lightning Network).

Task 1: Setup Zensical Site Configuration¶

Files: - Modify: requirements.txt - Create: zensical.toml

Interfaces: - Consumes: None - Produces: zensical.toml configuration file for compiling static docs.

  • Step 1: Update requirements.txt to add zensical

Add zensical to the requirements file. In requirements.txt, append the dependency: 

zensical

  • Step 2: Install dependencies locally

Run: 

conda run -n ezbitcoin pip install -r requirements.txt
Expected output: Successful installation of zensical package and its dependencies.

  • Step 3: Create zensical.toml configuration file

Create the file /opt/brainzhang/ezbitcoin/zensical.toml with the following content:

Task 2: Setup GitHub Actions workflow for deployment¶

Files: - Create: .github/workflows/deploy.yml

Interfaces: - Consumes: zensical.toml site configuration. - Produces: GitHub Actions deployment YAML configuration.

  • Step 1: Create deploy.yml workflow file

Create directory .github/workflows/ if it does not exist: 

mkdir -p .github/workflows
Then create file /opt/brainzhang/ezbitcoin/.github/workflows/deploy.yml with content: 
name: Deploy EzBitcoin Docs to GitHub Pages

on:
  push:
    branches:
      - main

permissions:
  contents: write

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout repository
        uses: actions/checkout@v4

      - name: Set up Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.12'

      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install -r requirements.txt
          pip install zensical

      - name: Build documentation site
        run: |
          zensical build

      - name: Deploy to GitHub Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./site
          cname: ezbitcoin.brainz.fun

  • Step 2: Commit deployment config
git add .github/workflows/deploy.yml
git commit -m "feat: add github actions workflow for Zensical deployment to github pages"
